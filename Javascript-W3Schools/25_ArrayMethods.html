'push()' Method

The 'push()' method append a new element to an array

'pop()' Method

The 'pop()' method removes the last element from an array

'join()' Method

The 'join()' method joins array elements into a string

In this example we have used "*" as a seperator between the elements

'splice()' Method

The 'splice()' method adds new elements to an array:

* array.splice(index, howmany, item1, ....)

* howmany: Optional. The number of items to be removed. If set to 0, no items will be removed

'toString()' Method

The 'toString()' method returns array elements as a 'string' seperated by ',' comma

'shift()' Method

The 'shift()' method removes the first element of an array (and shift all other elements to the left)

* 'Banana' has been removed

'unshift()' Method

The 'unshift()' method adds new element at the beginning of an array

* 'Banana' has been added

'slice()' Method

When the 'slice()' method is given two arguments, it selects array elements from the start argument, and upto (but not included) the last argument
