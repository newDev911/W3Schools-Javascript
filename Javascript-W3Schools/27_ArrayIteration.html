My Array

'forEach()' Method

Calls a function once for each array element

'map()' Method

The 'map()' method creates a new array with the results of calling a function for every array element

* The 'map()' method calls the provided function once for each array element

* The 'map()' doesn't execute the function for array elements without values

* The 'map()' doesn't change the original array

'filter()' Method

The 'filter()' method creates a new array with all array elements that passes the test

'reduce()' Method

The 'reduce()' method finds the sum of all array elements

'reduceRight()' Method

* The 'reduceRight() 'method reduces the array to a single value.

* The 'reduceRight()' method executes a provided function for each value of the array (from right-to-left).

'every()' Method

The 'every() 'method checks if all array values pass a test.

'some()' Method

The 'some() 'method checks if all array values pass a test.

'find()' Method

The 'find() 'method returns the value of the first element that pass a test (provided as a function).
